Commit all current changes and push to GitHub. Follow these steps:

1. Run `git status` to see all changes
2. Run `git diff` to review what changed
3. Run `git log --oneline -3` to match commit message style
4. Stage the relevant changed files using `git add <file1> <file2> ...` (do NOT use `git add -A`)
5. Write a clear, concise commit message summarizing the changes
6. Run `git commit -m "<commit message>"` to commit the staged changes
7. Run `git push` to push to the remote repository
8. Confirm success with the commit hash and remote URL
